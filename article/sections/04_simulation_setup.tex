\section{Simulation Setup}

The current manuscript scaffold is instantiated with the Rabot outdoor scenario defined in \texttt{scenarios/rabot.yaml}. The scene is derived from an \gls{osm} boundary around the KU Leuven Gent Campus Rabot environment. Candidate distributed-\gls{ap} positions are generated along building walls at low height, one rooftop proxy is reserved for the central reference strategy, and user mobility is simulated over the outdoor walkable area with mixed pedestrian and bicycle profiles.

The simulation workflow used by the paper matches the repository workflow:
\begin{enumerate}
    \item build the outdoor scene and walk graph from \gls{osm},
    \item generate wall-mounted candidate \glspl{ap} and the single rooftop proxy site,
    \item simulate user trajectories over the walkable outdoor area,
    \item compute path-based \gls{csi} and explicit \gls{sinr} terms,
    \item compare the central, fixed-distributed, and movable-distributed strategies,
    \item export reusable result files,
    \item generate manuscript figures in a separate postprocessing step.
\end{enumerate}

This section should stay descriptive and reproducible. It should document how the figures are produced, not what the figures mean.
